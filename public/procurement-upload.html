Required and optional columns

Required columns — analysis cannot run without these:

  • PO Number PO Number · PO No · Order Number · Purchase Order · Doc Number
  • Supplier Supplier · Vendor · Vendor Name · Creditor · Source
  • Line Amount Line Amount · Net Value · Extended Amount · Total Amount · Value

Delivery Date — at least one is recommended:

  • Due / Confirmed Date Due Date · Delivery Date · Promised Date · Confirmed Date · ETA · Scheduled Date
  • Need By Date Need Date · Need By · Required Date · Requested Date

Optional columns — enrich risk scoring when present:

  • Line Number / Line No
  • Item Code / Part Number / SKU / Material
  • Description / Item Description
  • Quantity Ordered / Qty Ordered
  • Quantity Received / Qty Received
  • Unit Price
  • Order Date / PO Date
  • Buyer / Purchasing Agent
  • Plant / Site / Location
  • Category / Commodity / Material Group

File format: Comma-delimited CSV (.csv) only, up to 5 MB. UTF-8 or Windows-1252 encoding. In Excel: Save As → CSV (Comma delimited) (*.csv) before uploading.

Common upload errors and fixes

Error: “missing required columns”

No recognised column found for PO Number, Supplier, or Line Amount. Check the accepted names in the column guide above, or download the sample CSV template as a reference.

Error mentions semicolons, tabs, or pipes

Non-comma delimiter detected. In Excel: Save As → CSV (Comma delimited) (*.csv) and re-upload.

Delivery date columns not detected

Rename your date column to one of the accepted names (Due Date, Delivery Date, ETA, etc.) and ensure dates use a standard format: YYYY-MM-DD, MM/DD/YYYY, or DD/MM/YYYY.

Error: file is too large

Max 5 MB. Narrow the export to a single plant, supplier group, or rolling date window and re-export.

Error: “no data rows”

File parsed but no rows found below the header. Confirm the export is not an empty or summary-only report.
