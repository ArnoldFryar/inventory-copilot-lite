Required and optional columns

Required column — import cannot proceed without this:

  • Part Number Part Number · Part No · Item Number · SKU · Material · Item Code

Recommended columns — enable data quality alerts:

  • Supplier Supplier · Vendor · Vendor Name · Manufacturer
  • Lead Time Lead Time Days · Lead Time · LT · Supplier Lead Time
  • Min / Max Quantities Min Qty · Min · MOQ · Max Qty · Max
  • Pack Multiple Pack Multiple · Order Multiple · Lot Multiple
  • ABC Class ABC Class · ABC · Velocity Class · ABC Code

Optional columns — enrich the register:

  • Part Description / Description
  • Secondary Supplier / Alternate Supplier / Backup Supplier
  • Supplier Part Number / Vendor Part No
  • Replenishment Method / Planning Method
  • Reorder Point / ROP / Safety Stock
  • Standard Pack / Package Size
  • Unit Of Measure / UOM
  • Unit Cost / Std Cost
  • Annual Usage / Annual Demand
  • Point Of Use / POU / Bin Location
  • Plant / Facility / Site
  • Commodity Class / Commodity
  • Notes / Comments

File format: Comma-delimited CSV (.csv) only, up to 5 MB. UTF-8 or Windows-1252 encoding. In Excel: Save As → CSV (Comma delimited) (*.csv) before uploading. Re-importing the same part numbers updates existing records — no duplicates are created.

Common import errors and fixes

Error: “missing required columns”

No recognised column found for Part Number. Check the accepted names in the column guide above, or download the sample PFEP template as a reference.

Error mentions semicolons, tabs, or pipes

Non-comma delimiter detected. In Excel: Save As → CSV (Comma delimited) (*.csv) and re-upload.

Some rows were skipped

Rows without a Part Number value are silently skipped. Ensure every row has a Part Number filled in.

Error: file is too large

Max 5 MB. Split the PFEP into smaller per-plant or per-commodity batches and re-import.

Duplicate parts after re-import?

No — re-importing the same Part Number updates the existing record. The register uses upsert semantics, so re-imports are safe and idempotent.
